# Todo Page

Simple Todo Kanban board

## 프로젝트 소개

### 내용

1. 칸반보드 형태의 간단한 Todo List
2. Drag/Drop을 활용한 Column, Task 관리

<br/>

## 프로젝트 실행 가이드

- 실행 방법 (2가지 중 택 1)
  > 1. ZIP 파일 다운로드 및 압축 풀기 후 코드 에디터로 실행
  > 2. 아래 커멘드를 이용한 실행

```bash
$ git clone https://github.com/H0onnn/LJH_ToDoPage.git
$ pnpm install
$ pnpm dev
```

<br/>
